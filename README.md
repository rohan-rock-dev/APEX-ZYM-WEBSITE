Where elite engineering meets human performance.

## About

APEX is a premium gym website built with React, Vite, TypeScript, GSAP, and Framer Motion. It features cinematic scroll animations, immersive section transitions, elite trainer profiles, interactive membership plans, and a full contact system.

## Tech Stack

React · TypeScript · Vite · GSAP · Framer Motion · Tailwind CSS · Three.js · Lucide React

## Getting Started

### Step 1 — Clone the repository

git clone https://github.com/yourusername/apex-gym.git

### Step 2 — Navigate into the project folder

cd apex-gym

### Step 3 — Install all dependencies

npm install

### Step 4 — Start the development server

npm run dev
